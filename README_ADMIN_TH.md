# Admin Panel รุ่นจัดระเบียบใหม่

ชุดนี้อิงจากไฟล์ก่อนเพิ่ม admin แล้วเพิ่มหน้าจัดการแอดมินให้ใหม่แบบไม่ให้ระบบหลง

## จุดเด่น
- ปุ่มภาษาไทยทั้งหมด
- มีคำอธิบายวิธีใช้ใน panel
- ทางเดินชัดเจน: หน้าหลักแอดมิน -> จัดการพาเนล / จัดการข้อมูลเควส
- ฝั่งเควสใช้หลัก "เลือกเควสก่อน แล้วค่อยดูข้อมูลของเควสนั้น"
- ตอนบอทเปิด จะ deploy แค่ admin panel
- พาเนลผู้เล่นให้จัดการจากหน้าแอดมินเอง เพื่อลด warning ตอน config ยังไม่ครบ

## วิธีใช้
1. ตั้งค่า `QUEST_ADMIN_PANEL_CHANNEL`
2. รันบอท
3. หรือรัน `node src/scripts/createAdminPanel.js`
4. เข้าไปกดใน panel:
   - จัดการพาเนล
   - จัดการข้อมูลเควส

## หมายเหตุ
- เวอร์ชันนี้ทำฝั่งดูข้อมูลและจัดทางเดินก่อน
- เมนูแก้ไขจริงยังเป็น placeholder
